#Auto Intensity Street Light
Intern ID: CITS3610
Full Name: Sanjay S 
Project Name:
Auto Intensity Street Light
Project Scope :
The Auto Intensity Street Light project uses an Arduino and an LDR (Light Dependent Resistor) to automatically control the brightness of a street light based on the surrounding light intensity. This helps reduce power consumption, improves energy efficiency, and provides automatic operation without manual intervention.
